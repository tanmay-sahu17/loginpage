# Login Page

A modern, responsive login page built with HTML, CSS, and JavaScript.

## Features

- **Modern Design**: Clean, professional interface with gradient background and glassmorphism effects
- **Responsive Layout**: Fully responsive design that works on desktop, tablet, and mobile devices
- **Form Validation**: Real-time client-side validation for email and password fields
- **Interactive Elements**: 
  - Password visibility toggle
  - Remember me functionality
  - Loading states with spinner animation
  - Success/error message display
- **Social Login Options**: Buttons for Google and Facebook authentication (demo functionality)
- **Accessibility**: Keyboard navigation support and proper focus management
- **Fallback Icons**: Works even when CDNs are blocked, with emoji fallbacks

## Demo Credentials

- Password: `password123`

Or any valid email format with a password of at least 6 characters.

## File Structure

- `index.html` - Main HTML structure
- `styles.css` - CSS styling with responsive design
- `script.js` - JavaScript for form validation and interactivity

## Usage

1. Open `index.html` in a web browser
2. Enter email and password
3. Click "Sign In" to test the login functionality
4. Try the social login buttons and other interactive features

## Browser Compatibility

Compatible with all modern browsers including Chrome, Firefox, Safari, and Edge.
